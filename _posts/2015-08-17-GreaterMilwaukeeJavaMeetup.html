I've started hosting the Greater Milwaukee Java Meetup. Join us bi-monthly to learn about new Java technologies and see them in action.

The meetup location changes to get the highest number of Milwaukee developers interested and involved. Past locations have included FedEx SmartPost / Services in Brookfield, and PKWARE in the 3rd ward.
