Welcome

The web is a jumbled, and frightening place for one's online presence. To combat my many fragmented projects and profiles, I've created this simple site to tie them all together in one place. Here you will find my thoughts, projects, and any other snippets of web presence I may have.

To make this initiative simple and free I decided to build the site with the most basic tools available. The site runs for free on the GitHub Pages platform. The site is statically generated by the Jeckyll framework, and uses the Twitter Bootstrap css framework. The LinkedIn, StackOverflow, GitHub, and Twitter JavaScript APIs are utilized as well.
